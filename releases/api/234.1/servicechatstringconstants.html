ServiceChat Strings

List of string constants associated with the Snap-ins SDK Live Agent Chat framework.

  • ServiceCloud.Chat.Dialog.General.Title

    Description

    Title of many alert dialogs and views.

    Default Value

    Live Chat

  • ServiceCloud.Chat.Dialog.General.Cancel

    Description

    Negative option in generic alert dialogs.

    Default Value

    Cancel

  • ServiceCloud.Chat.Dialog.General.Confirm

    Description

    Affirmative option in generic alert dialogs.

    Default Value

    OK

  • ServiceCloud.Chat.Dialog.PreChat.Content

    Description

    String presented to mobile user above a number of text fields that require input before submitting a chat request.

    Default Value

    Just a few things before we connect you with an expert…

  • ServiceCloud.Chat.Dialog.PreChat.Confirm

    Description

    Label of button for mobile user to press once they have completed necessary text fields and are ready to chat with an agent.

    Default Value

    Chat with an Agent

  • ServiceCloud.Chat.Dialog.PreChat.Confirm.Disabled

    Description

    Label of button for mobile user to inform the user the chat with agent button is disabled.

    Default Value

    Chat with an agent disabled

  • ServiceCloud.Chat.Dialog.EndSession.Title

    Description

    Title of end session alert dialog presented when a mobile user attempts to end the session.

    Default Value

    End Session

  • ServiceCloud.Chat.Dialog.EndSession.Content

    Description

    Message body in end session alert dialog presented when a mobile user attempts to end the session.

    Default Value

    Are you sure you want to end the chat?

  • ServiceCloud.Chat.Dialog.EndSession.Confirm

    Description

    Affirmative option in end session alert dialog presented when a mobile user attempts to end the session.

    Default Value

    End

  • ServiceCloud.Chat.Dialog.Photo.TakePicture

    Description

    Option presented to mobile user for sending a file to an agent by taking a new photo with the camera.

    Default Value

    Take Photo

  • ServiceCloud.Chat.Dialog.Photo.Library

    Description

    Option presented to mobile user for sending a file to an agent by selecting a previously captured image from the device’s photo library.

    Default Value

    Photo Library

  • ServiceCloud.Chat.Dialog.NoAgents.Content

    Description

    Alert message body displayed to mobile user when no agents are available to chat with.

    Default Value

    Sorry, no agents are available now. Please try again later.

  • ServiceCloud.Chat.Dialog.SessionTerminated.Title

    Description

    Alert title displayed to mobile user when their chat session has ended.

    Default Value

    Session Ended

  • ServiceCloud.Chat.Dialog.SessionTerminated.Content

    Description

    Alert message body displayed to mobile user when an agent has ended the chat session.

    Default Value

    The agent ended the session.

  • ServiceCloud.Chat.Dialog.PreChat.SubmissionError

    Description

    Label of the submit button on prechat in an error condition

    Default Value

    Submission Error

  • ServiceCloud.Chat.Dialog.PreChat.SubmissionInProgress

    Description

    Label of the submit button on prechat when submit request is in progress

    Default Value

    Submit In Progress

  • ServiceCloud.Chat.Dialog.PreChat.SubmissionComplete

    Description

    Label of the submit button on prechat when submit is completed

    Default Value

    Submission Complete

  • ServiceCloud.Chat.Dialog.PreChat.Picklist.SelectAValue

    Description

    Default picklist option in prechat picklist

    Default Value

    Select a value...

  • ServiceCloud.Chat.Dialog.Popover.TapToMinimize

    Description

    Label inside tooltip to show customer where to press to minimize window

    Default Value

    Tap to minimize window

  • ServiceCloud.Chat.Dialog.Popover.GotIt

    Description

    Label of button inside tooltip to dismiss tooltip

    Default Value

    GOT IT

  • ServiceCloud.Chat.Status.Connecting

    Description

    Status message displayed to mobile user when connecting to chat servers.

    Default Value

    Connecting...

  • ServiceCloud.Chat.Status.ConnectingLong

    Description

    Status message (in long form) displayed to mobile user when connecting to chat servers.

    Default Value

    Connecting to chat...

  • ServiceCloud.Chat.Status.Active.Now

    Description

    Status message displayed to mobile user when agent is currently active in the chat session.

    Default Value

    active now

  • ServiceCloud.Chat.Status.Inactive.SingleMinute

    Description

    Status message displayed to mobile user when agent has been inactive for a minute.

    Default Value

    active 1 min ago

  • ServiceCloud.Chat.Status.Inactive.MultipleMinutes

    Description

    Status message displayed to mobile user when agent has been inactive for a specific number of minutes.

    Default Value

    active %ld mins ago

  • ServiceCloud.Chat.Status.Inactive.Hour

    Description

    Status message displayed to mobile user when agent has been inactive for over an hour.

    Default Value

    active over an hour ago

  • ServiceCloud.Chat.Status.Waiting

    Description

    Status message displayed to mobile user when waiting for agent to connect to a session.

    Default Value

    Waiting for Agent

  • ServiceCloud.Chat.Status.PleaseWait

    Description

    Message displayed when waiting to be routed to an available agent.

    Default Value

    We’ll be with you as soon as an agent is available.

  • ServiceCloud.Chat.Status.QueuePosition

    Description

    Status message displayed above an integer value representing the number of people ahead of you in chat queue.

    Default Value

    Your place in line

  • ServiceCloud.Chat.Status.QueuePosition.Format

    Description

    Format string for displaying the user’s queue position, e.g. “#5”.

    Default Value

    #%ld

  • ServiceCloud.Chat.Status.EstimatedWait.Title.Minimized

    Description

    Status message displayed in the minimized state above the time label indicating the estimated wait time for a chat in the queue.

    Default Value

    Wait time:

  • ServiceCloud.Chat.Status.EstimatedWait.Title.FullScreen

    Description

    Status message displayed in the fullscreen state above the time label indicating the estimated wait time for a chat in the queue.

    Default Value

    Your estimated wait time is:

  • ServiceCloud.Chat.Status.EstimatedWait.Unknown.Minimized

    Description

    Estimated wait message to show in the minimized state when we do not know how long the user has in the queue. This may happen near the end of the queue when we expect they would have connected already.

    Default Value

    Waiting for an agent

  • ServiceCloud.Chat.Status.EstimatedWait.Unknown.FullScreen

    Description

    Estimated wait message to show in the full screen state when we do not know how long the user has in the queue. This may happen near the end of the queue when we expect they would have connected already.

    Default Value

    We’ll be with you as soon as an agent is available.

  • ServiceCloud.Chat.Status.EstimatedWait.Title.Unknown.FullScreen

    Description

    Estimated wait title to show in the full screen state when we do not know how long the user has in the queue.

    Default Value

    Please hang tight…

  • ServiceCloud.Chat.Status.EstimatedWait.Excessive.Minimized

    Description

    Estimated wait message to show in the minimized state when the estimated wait time is larger than the configured maximum.

    Default Value

    Extended wait time

  • ServiceCloud.Chat.Status.EstimatedWait.Excessive.FullScreen

    Description

    Estimated wait message to show in the full screen state when the estimated wait time is larger than the configured maximum.

    Default Value

    We’re experiencing longer than expected wait times.

  • ServiceCloud.Chat.Status.EstimatedWait.Title.Excessive.FullScreen

    Description

    Estimated wait title to show in the full screen state when the estimated wait time is larger than the configured maximum.

    Default Value

    Please hang tight…

  • ServiceCloud.Chat.Status.Ended

    Chat Ended

  • ServiceCloud.Chat.Status.Failed

  • ServiceCloud.Chat.Status.Cancelled.NoAgentsDetailed

  • ServiceCloud.Chat.Status.Cancelled.UserDetailed
