ServiceCore Strings

List of string constants that are shared across all Snap-ins SDK frameworks.

  • ServiceCloud.Common.NoConnectionErrorMsgTitle

    Description

    Title displayed when there is no internet connection

    Default Value

    Unable to connect...

  • ServiceCloud.Common.NoConnectionErrorMsgDescription

    Description

    Description displayed when there is no internet connection

    Default Value

    Please ensure you are online

  • ServiceCloud.Common.Error.UserSessionExpiredOrInvalidSession

    Description

    Description displayed when user account has expired

    Default Value

    The session ID or OAuth token used has expired or is invalid.

  • ServiceCloud.Common.Error.UserSessionRequestRefused

    Description

    Description displayed when user does not has sufficient access.

    Default Value

    The request has been refused. Verify that the logged-in user has appropriate permissions.

  • ServiceCloud.Common.Error.ResourceNotFound

    Description

    Description displayed when there is an Authentication Failure

    Default Value

    The requested resource could not be found.

  • ServiceCloud.Common.Error.GenericErrorMsgTitle

    Description

    Main Message displayed for generic error

    Default Value

    Oops. Something went wrong...

  • ServiceCloud.Common.Error.GenericErrorMsgDescription

    Description

    Second line of Message displayed for generic error condition

    Default Value

    Please try again or contact the administrator

  • ServiceCloud.Common.NoAccessErrorMsgTitle

    Description

    Title displayed when the user does not have access

    Default Value

    Unable to access

  • ServiceCloud.Common.Network.Reachability.ConnectionLost

    Description

    Message displayed when the internet connection is lost

    Default Value

    Connection Lost

  • ServiceCloud.Common.Network.Reachability.Connected

    Description

    Message displayed when the internet connection is re-established

    Default Value

    Connected!

  • ServiceCloud.Common.Network.Reachability.Reconnecting

    Description

    Message displayed when attempting to reconnect.

    Default Value

    Having trouble connecting. We'll keep trying.

  • ServiceCloud.Common.User.Text.Input.Accessibility

    Description

    Accessibility string for the user text input in the Chat and Cases view controller

    Default Value

    User text input

  • ServiceCloud.Common.Send.Button.Accessibility

    Description

    Accessibility string from the send button in the Chat and Cases view controller

    Default Value

    Send message to agent
